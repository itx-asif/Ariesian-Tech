### 1. Audio Player
The first project is an Audio Player. This project allows users to play, pause, and skip audio tracks with a clean and user-friendly interface.

### 2. Image Gallery
The second project is an Image Gallery. This project showcases images in a grid format with lightbox functionality.

- **Project:** [Image Gallery](https://itx-asif.github.io/Ariesian-Tech/image-gallery/)
- **Description:** An interactive image gallery created with HTML, CSS, and JavaScript. Users can click on images to view them in a larger format, and it includes responsive design for mobile devices.

### 3. Tic-Tac-Toe
The third project is a Tic-Tac-Toe game. This project allows two players to play the classic game in a web browser.

- **Project:** [Tic-Tac-Toe](https://itx-asif.github.io/Ariesian-Tech/tictoctoe/)
- **Description:** A Tic-Tac-Toe game developed using HTML, CSS, and JavaScript. Players can take turns marking their spots, and the game determines the winner or if it's a draw.

### 4. Stopwatch
The fourth project is a Stopwatch. This project allows users to time events with start, stop, reset, and lap functionality.

- **Project:** [Stopwatch](https://itx-asif.github.io/Ariesian-Tech/stopwatch/)
- **Description:** A functional stopwatch built with HTML, CSS (using Tailwind), and JavaScript. It features lap recording, a clean interface, and responsive design.
